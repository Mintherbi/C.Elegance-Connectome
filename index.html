C. elegans Connectome Visualization
Interactive Neural Network Explorer
Advisor: Catherine Griffiths
Abstract

This project presents an interactive visualization of the C. elegans connectome, the complete neural network of the nematode worm. Using D3.js, we create a dynamic network graph where each neuron is represented as a node and synaptic connections as edges. The visualization incorporates spatial positioning data and color-codes connections based on neurotransmitter types, providing an intuitive way to explore the neural architecture of this model organism.

Motivation

C. elegans is the only organism with a completely mapped nervous system, making it an ideal model for understanding neural connectivity patterns. This visualization tool helps researchers and students explore the complex relationships between neurons, identify important network hubs, and understand how spatial organization relates to functional connectivity in the nervous system.
